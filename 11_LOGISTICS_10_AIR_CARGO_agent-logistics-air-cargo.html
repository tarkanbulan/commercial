AGENT_LOG_AIRCARGO — Air Cargo Integrity & Compliance Module

Market: Global $150-200B. UK £10-15B. Heathrow #1 EU cargo. Key UK Airports: LHR (Heathrow), EMA (East Midlands), STN (Stansted), BHX

1. Mission

Air cargo integrity: AWB fraud, dangerous goods misdeclaration, pharma cold chain, screening compliance, and cargo theft detection.

2. Detection Architecture

2.1 Air Waybill Fraud

  • A1 — Phantom AWB: AWB issued but no cargo received/loaded
  • A2 — Rate Manipulation: Actual freight rate vs. AWB declared rate (profit skimming)
  • A3 — Weight Manipulation: Declared chargeable weight vs. actual. DIM factor manipulation

2.2 Dangerous Goods Compliance

  • D1 — Misdeclaration: Shipper declares non-DG but cargo is DG (IATA DGR non-compliance)
  • D2 — Packing Instruction: DG packed against relevant packing instruction
  • D3 — Hidden DG: Undeclared DG in passenger baggage (UPSC screening)

2.3 Pharma / Cold Chain

  • P1 — Temperature Excursion: Active/passive container temp data during air transit
  • P2 — Time-Critical: RUSH/priority shipment handling time compliance
  • P3 — CEIV Pharma: IATA CEIV certification compliance audit

2.4 Screening / Security

  • S1 — ACC3 Compliance: Known consignor / regulated agent screening adherence
  • S2 — Piece Count: Security screened pieces vs. loaded pieces reconciliation
  • S3 — X-Ray Competency: Screener image evaluation timing and accuracy

3. Proprietary Indicators

Indicator Description Threshold
AWB_W chargeable_weight - actual_weight >10% = MANIPULATION
DG_RISK declared_dg - actual_dg_cargo mismatch = CRITICAL
SCREEN_C screened_pieces - loaded_pieces >0 = BREACH
