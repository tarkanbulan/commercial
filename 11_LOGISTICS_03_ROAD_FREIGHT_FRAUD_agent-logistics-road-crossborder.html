AGENT_LOG_CROSSB — Cross-Border Road Freight Compliance Module

Regulatory: EU Cabotage rules, UK-EU TCA, TIR Convention, ATLAS/ NCTS

1. Mission

Cross-border road freight compliance: cabotage violations, customs fraud, fuel duty evasion, and transit document manipulation detection.

2. Detection Architecture

2.1 Cabotage Compliance

  • C1 — Domestic Trip Count: Non-resident carrier performing >3 domestic trips in 7 days = violation
  • C2 — Cool-Off Period: Cabotage trips followed by insufficient cross-border return
  • C3 — Load Verification: Cross-border load vs. domestic load pattern analysis

2.2 Customs Declaration Integrity

  • T1 — TIR Carnet: Missing stamps, extended transit times, seal manipulation
  • T2 — HS Code: Misclassification for duty/tariff advantage
  • T3 — Value Manipulation: Under-declared value for customs valuation

2.3 Fuel Duty Fraud

  • F1 — Diesel Laundering: Fuel consumption vs. mileage anomalies
  • F2 — Red Diesel: Rebated fuel use in road vehicles (UK HMRC fraud)
  • F3 — Cross-Border Fuel: Fuel purchased in low-duty jurisdiction vs. consumption geography

3. Proprietary Indicators

Indicator Description Threshold
CABOTAGE_N domestic_trips_in_7d >3 = VIOLATION
TRANSIT_T actual_transit_time - allowed_time >20% = INVESTIGATE
FUEL_J fuel_jurisdiction vs. actual_operating_area mismatch = DUTY_EVASION
