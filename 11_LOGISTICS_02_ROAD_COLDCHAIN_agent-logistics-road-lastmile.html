AGENT_LOG_LASTMIL — Last-Mile Delivery Forensics Module

Market: Global last-mile delivery $200B+. UK parcel market £15B+. Key Players: DHL, UPS, FedEx, Royal Mail, Evri, DPD, Amazon Logistics

1. Mission

Detect last-mile delivery fraud: fake delivery claims, time window violations, package theft, and route manipulation.

2. Detection Architecture

2.1 Delivery Proof Manipulation

  • D1 — Geo-Fence Violation: Delivery marked complete but driver GPS outside delivery address
  • D2 — Photo Forensics: Delivery photo metadata (timestamp, GPS) vs. claimed delivery
  • D3 — Signature Abuse: Recipient signature consistency for same address across deliveries

2.2 Time Window Compliance

  • T1 — Slot Adherence: Actual delivery time vs. promised window
  • T2 — Failed Delivery Integrity: "Nobody home" claims vs. doorbell/camera data
  • T3 — Re-Delivery Fraud: Second delivery attempt billed but first delivery actually succeeded

2.3 Driver Performance Forensics

  • P1 — Route Deviation: Planned route vs. actual. Detours = personal errands/time theft
  • P2 — Idle Time: Excessive stationary periods during delivery route
  • P3 — Package Tampering: Package damage rates by driver vs. route baseline

3. Proprietary Indicators

Indicator Description Threshold
GEO_FENCE deliver_gps - customer_address_gps >100m = FAKE_DELIVERY
SLOT_D actual_delivery - promised_window_end >0 = LATE
ROUTE_D planned_route_miles - actual_miles >10% = THEFT_OF_TIME
