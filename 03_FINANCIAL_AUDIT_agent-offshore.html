AGENT_OFFSHORE — Offshore & Hidden Ownership Detection

Source: T2SAIM Commercialization Framework v03 (§8.2 — Global Intelligence Journalism) Context: Panama Papers, FinCEN Files, Dubai Unlocked — terabyte-scale international data leaks

1. Mission

Process terabyte-scale offshore data leaks through structural anomaly detection. Map shell company board compositions, name-address similarities, and actual beneficial owners autonomously.

2. Detection Architecture

2.1 Shell Company Detection

  • S1 — Board Composition Anomaly: Same directors across hundreds of shell companies
  • S2 — Address Clustering: Thousands of companies registered at same address
  • S3 — Capital Structure Testing: Minimal capitalization with massive transaction volumes

2.2 Beneficial Owner Identification

  • O1 — Name Similarity Mapping: Variations of same name across jurisdictions
  • O2 — Multi-Layer Ownership Tracking: Through 5+ layers of nominee directors
  • O3 — Professional Enabler Detection: Same law firm/accountant appearing across shell network

2.3 Cross-Jurisdiction Pattern Recognition

  • J1 — Tax Haven Flow Mapping: Money movement between secrecy jurisdictions
  • J2 — Trade Mispricing Detection: Export/import value discrepancies
  • J3 — Timeline Coherence: Company creation → transaction → dissolution patterns

3. Commercial Model

  • RaaS (Research-as-a-Service): Subscription for investigative journalism organizations
  • Platform Integration: T2SAIM as analytical meta-layer on existing investigative platforms (OCCRP, ICIJ)

4. Integration

  • ICIJ Offshore Leaks Database cross-reference
  • OCCRP Aleph platform compatibility
  • OpenCorporates / OpenSanctions data feeds
