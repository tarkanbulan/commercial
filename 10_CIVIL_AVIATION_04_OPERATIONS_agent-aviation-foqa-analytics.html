AGENT_AVIATION_FOQA — Flight Data Monitoring Analytics Module

1. Mission

Deep structural analysis of flight data recorder (FDR/QAR) data. Detect exceedance concealment, fuel fraud, approach instability patterns, and safety culture from aggregate flight data.

2. Detection Architecture

2.1 Exceedance Pattern Analysis

  • E1 — Exceedance Concealment: Exceedances in raw data not logged in pilot reports
  • E2 — Recurring Non-Compliance: Same aircraft/pilot/captain repeating same exceedance type
  • E3 — Trending: Exceedance rates increasing over time = safety culture indicator

2.2 Approach/Landing Analysis

  • A1 — Unstable Approach: Speed, altitude, configuration deviations from stabilized approach criteria
  • A2 — Hard Landing Detection: Vertical acceleration + sink rate cross-reference
  • A3 — Go-Around Analysis: Go-around decision timing and altitude

2.3 Fuel Efficiency Forensics

  • F1 — APU Usage Anomaly: APU run time vs. ground power availability
  • F2 — Fuel Tankering: Fuel loaded beyond needed for sector
  • F3 — Optimum Altitude Deviation: Cruising altitude vs. optimum for weight/conditions

3. Proprietary Indicators

Indicator Description Threshold
EXCEED_C raw_exceedances - reported_exceedances >0 = CONCEALMENT
STABLE_R unstable_approaches / total_approaches >5% = RISK
FUEL_D actual_fuel_burn - expected_burn >5% = ANOMALY
