AGENT_LOG_PORT — Port Operations & Customs Forensics Module

1. Mission

Port operations integrity: customs declaration fraud, freeport zone anomalies, cargo exam manipulation, and port state compliance.

2. Detection Architecture

2.1 Customs Declaration Fraud

  • C1 — HS Code Misclassification: Systematic miscoding for duty advantage
  • C2 — Value Under-declaration: Declared value vs. transaction database
  • C3 — Origin Fraud: False origin claims for preferential tariff

2.2 Freeport / FTZ Monitoring

  • F1 — Goods Movement: Cargo entered freeport but never exited (domestic diversion)
  • F2 — Processing Claims: Manufacturing/processing in freeport vs. actual capability
  • C3 — Bond Manipulation: Customs bond expiry, under-bonding

2.3 Port State Control Forensics

  • P1 — Inspection Targeting: Vessels with deficiency patterns not inspected
  • P2 — Deficiency Correlation: Same deficiencies across fleet/owner/flag
  • P3 — Detention Analysis: Port state detention validity

2.4 Container Exam Integrity

  • E1 — Screening Rates: Scanner/NII exam rates vs. risk profile
  • E2 — Exam Result Manipulation: Positive finding suppressed at local level
  • E3 — Selectivity Override: Customs override of system-selected exams

3. Proprietary Indicators

PSC_D flagged_deficiencies - actual_condition >3 = SYSTEMIC
