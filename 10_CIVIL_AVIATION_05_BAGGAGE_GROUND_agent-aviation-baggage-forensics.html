AGENT_AVIATION_BAG — Baggage Handling Integrity Module

Market: 4.5B bags checked annually, 0.5% mishandled (~22M bags), $2.5B annual cost Regulatory: IATA Resolution 753 (bag tracking at 4 key points), Montreal Convention (liability) Key Players: SITA, Amadeus, Vanderlande, Daifuku, BCS Group

1. Mission

Baggage handling system structural integrity testing. Detect tracking gaps, mishandling root causes, theft, and systemic failures in baggage operations.

2. Detection Architecture

2.1 RFID Tracking Integrity

  • R1 — Scan Gap Detection: Missing scan at any of IATA 753 mandatory points (check-in, load, transfer, arrival)
  • R2 — Timing Anomaly: Bag scanned at implausible speed between points
  • R3 — Handoff Failure: Bag transferred between carriers but scan chain breaks at handoff

2.2 Mishandling Root Cause Analysis

  • M1 — Transfer Time: Connection time vs. minimum connection time (MCT) gap
  • M2 — Flight-Specific: Particular flights with higher mishandling rates = systemic problem
  • M3 — Station-Correlated: Particular airports/stations with chronic failure patterns

2.3 Baggage Theft/Tampering

  • T1 — Weight Change: Bag weight at check-in vs. arrival. Difference = tampering indicator
  • T2 — Route Deviation: Bag traveled through non-standard path
  • T3 — Access Pattern: Unauthorized baggage system access correlated with loss

3. Proprietary Indicators

Indicator Description Threshold
SCAN_C completed_scans / required_scans <1.0 = GAP
TIMING_D actual_transfer_time - min_required <MCT = RISK
WEIGHT_D arrival_weight - checkin_weight >1kg = TAMPERING
